#DesktopPasswordGenerator

Make A Desktop and Online Passwordgenerator to create your one secure Password and Learn to use js and Electron.

Steps:
  1-Build a Js Passwordgenerator
  2-Download npm node and start to use Electron
  3-Transfair your Site to Electron
  4-(Optional) Add features like -> Create everywere your password like atome it hase for open a Project Or Make some nice Animations OR Copy it directly to Clipboard. Feal Free..
  5-Export it for Win, Linux and Mac as .exe and Or .app
